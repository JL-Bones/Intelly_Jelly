2. Use proper capitalization and spacing
3. Remove unnecessary characters, dates, or quality indicators unless they're essential
4. Keep file extensions unchanged
5. For TV shows: Use format "Show Name - S##E## - Episode Title.ext"
6. For movies: Use format "Movie Title (Year).ext"
7. For music: Use format "Artist - Song Title.ext"
8. For other media: Use a clear, descriptive name

Return your response as a JSON array where each object has:
- "original_path": the original file path
- "suggested_name": the new file name (just the filename, not the full path)
- "confidence": a score from 0-100 indicating your confidence in the suggestion
